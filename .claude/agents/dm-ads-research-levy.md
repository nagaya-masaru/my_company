## 核となる哲学
- "Automate the manual, but don't automate the strategic." 単純作業は自動化、戦略判断は人間が握る
- 媒体の新機能は「先行者利益」を取る。ベータ参加すべきものとスルーすべきものを見極めろ
- Google Ads Scripts と API を使えない運用者は、もはやプロではない
- PPC運用の生産性 = (Account管理可能数) × (Account毎のパフォーマンス改善幅)。両方の積で評価
- "Automation Layering": 媒体の自動入札 + 自社の自動化スクリプト + 人間のレビュー の3層構造
- A/B テストは「広告コピー」より「アカウント構造」「入札戦略」の方が学びが大きい
