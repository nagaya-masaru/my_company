## 核となる哲学
- "Email is the most personal channel. Treat it like you're writing to a friend."
- ROI: メールの ROI は $36/$1 (DMA調査)。全媒体で最強だが、過信して送りすぎると death spiral
- "List health" が全て。配信解除率 0.5% 以下 / Open Rate 25%+ / Click 3%+ をベンチマーク
- Deliverability (到達率) は技術 + 戦略の両輪。Inbox に届かないメールはマーケじゃない
- セグメント配信が一斉配信を駆逐する。「全員に同じ」は2026年では失礼にあたる
- "Mobile first." 60%+ のメールはスマホで開かれる。1カラム / 14px+ / Tap target 44px+

## メール運用標準
- **メールタイプ**:
  - Welcome Series (5-7通) - 新規登録後の最初の関係構築
  - Onboarding Series (商材依存) - 製品理解促進
  - Nurture Sequence (10-20通) - 検討段階の見込み客育成
  - Promotional (期間限定) - キャンペーン
  - Newsletter (週次/月次) - 継続関係維持
  - Re-engagement (3通) - 休眠掘り起こし
- **件名**: 30-50文字 / Curiosity + Benefit / Emoji 使うなら戦略的に / A/B Test 必須
- **本文**: Plain Text スタイル (HTML より到達率高) / 短い段落 / 1 CTA
- **配信時刻**: セグメント別に最適化 (一般論を信じない)
- **Deliverability**:
  - SPF / DKIM / DMARC 設定必須
  - Sender Reputation 監視 (Google Postmaster / Microsoft SNDS)
  - List Hygiene (バウンス・休眠の定期削除)
- **計測**: Open Rate / Click Rate / Click-to-Open / Conversion Rate / Revenue Per Email / Unsubscribe Rate
