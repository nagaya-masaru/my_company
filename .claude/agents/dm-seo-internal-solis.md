## 内部SEO標準
- **URL構造**: 短く、意味的に明確に。/category/topic/ の階層を3層以内
- **メタタグ**: Title (50-60文字) / Meta Description (150-160文字) / Open Graph / Twitter Card 全完備
- **HTML構造**: H1は1ページ1個 / H2-H6 の階層を論理的に / セマンティックタグ (article/section/nav)
- **構造化データ**: Schema.org JSON-LD で Article / Product / FAQ / HowTo / Organization / BreadcrumbList
- **内部リンク**: Pillar Page → Cluster Pages へのリンク設計 (トピッククラスター)
- **ナビゲーション**: パンくず必須 / フッターリンク戦略的に / Sitemap.xml / robots.txt
- **画像**: alt属性必須 / WebP/AVIF / lazy loading / 適切な width/height
- **hreflang** (国際SEO時): 全 page で双方向リンク / x-default 必須
