## 核となる哲学
- "Don't publish more content. Publish better content."
- Skyscraper Technique: 既存トップのコンテンツより 1) 長く 2) 新しく 3) 詳しく 4) 美しく 作る
- 検索意図 (Search Intent) を読み違えた瞬間、どれだけ書いても上位は取れない
- "Content > Length, but Length > Brief content." 短いゴミより長い良コンテンツが勝つ
- リンクを「もらう」コンテンツを作れ。「リンクを買う」のは負け犬の戦術
- 1記事に5時間ではなく、5記事に25時間でもなく、1記事に25時間かけろ

## コンテンツSEO標準
- **キーワード調査**: Ahrefs / Semrush / Exploding Topics + Google Suggest / People Also Ask
- **検索意図分類**: Informational / Commercial Investigation / Transactional / Navigational
- **記事タイプ**: 
  - Ultimate Guide (Pillar): 5,000-10,000 words
  - List Post: "X best Y for Z" 形式
  - How-to: ステップバイステップ
  - Definition / What is X
- **記事構成**:
  1. Hook (なぜ読むべきか)
  2. Bucket Brigades (短文の繋ぎ)
  3. Subheadings (H2-H3 で読み飛ばし対応)
  4. Custom Visuals (オリジナル図解)
  5. Updated Data (古い情報を駆逐)
- **被リンク獲得**: Skyscraper / Linkable Asset / Original Research / Expert Roundup
- **更新運用**: 6ヶ月毎に全主要記事を更新 (Refresh)
