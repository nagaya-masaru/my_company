## 使い方

このディレクトリで `claude` を起動すれば、メインClaudeがCEOの代行として動く。

例えばCEOが
- 「営業戦略について Belfort と Kotler の意見を聞きたい」
- 「この新規事業を取締役会にかけてくれ」
- 「秘書室、来週のスケジュールを整理して」

と発話すれば、対応するサブエージェントが召喚される。
