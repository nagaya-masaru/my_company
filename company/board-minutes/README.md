# 取締役会議事録

`company/board-minutes/YYYY-MM-DD-{議題スラッグ}.md` の命名で保存。

## フォーマット

```markdown
# 取締役会議事録 — YYYY-MM-DD

## 議題
{何を Stop/Go するか}

## 出席
全員 (Musk / Gates / Page / Zuckerberg / Bezos / Huang / 孫) + CEO + 秘書室 (記録)

## 各取締役発言
### Elon Musk
{発言内容}
### Bill Gates
...

## 投票結果
Go ○票 / Stop ○票 / 条件付きGo ○票

## 決議
{採択された結論と付帯条件}

## CEO最終判断
{CEOがそのまま採択 / 修正 / 覆した場合の根拠}
```

秘書室 [[ceo-secretary-shidara]] が記録・保管責任を負う。
