# 取締役会議事録 2026-05-27 No.002

## 議題
宮野友樹氏との共同 AI Roll Up 事業の正式推進開始に関する Stop/Go 決議

## 承認を求めた範囲
1. 宮野氏との初動MTG (2026-06-03) の正式実施
2. 商標「Reboot Japan」シリーズ即時出願 (古美門案)
3. Phase 0 (~2026Q4) は本業集中、Roll Up は Partnership Charter 策定と士業ファネル構築のみ。買収実行は2027Q3以降
4. Hinton 提案「CFO代行 → CFO Copilot」呼称変更の全社適用
5. Attia 提案「CEO 健康指標を撤退条件の正式 KPI に組込み」

## 各取締役の見解と投票

### Elon Musk
First principles で人類への貢献と速度を評価。Conditional Go (速度倍速化条件)。Phase 0 を Q4 までは遅い、Partnership Charter は6月中固め、商標は即取得。CFO Copilot 名称・健康KPI 全面支持。

**投票: Conditional Go**

### Bill Gates
宮野氏×士業ネットワークは defensible advantage と評価。Phase 0 本業集中も Marks サイクル判定と整合的。ただし Partnership Charter で収益分配・知財帰属・宮野氏Exit条項を明確化することを強条件。

**投票: Go (Partnership Charter条項明確化条件)**

### Larry Page
30社を Knowledge Graph で繋ぐ構造を真の10xと評価。技術アーキテクチャをUnix哲学的にエレガントと評価。CFO Copilot 名称も支持。

**投票: Go**

### Mark Zuckerberg
A/B/C三層+共通基盤+クロスセルのプラットフォーム化として完璧と評価。最初の100顧客 (A層) で Product/Market Fit が先という観点で Phase 0 スローダウンを支持。

**投票: Go**

### Jeff Bezos
本日の決議は two-way door (パートナーシップ開始) であり Go。買収実行 (one-way door) を2027Q3以降にする規律を評価。Customer Obsession の観点で宮野氏士業ネットワークの「信頼の橋」が中小経営者AIアレルギーを越える鍵と評価。

**投票: Go**

### Jensen Huang
30日先と30年先を同時評価。30社統合は AI推論基盤30箇所展開と等しく、NVIDIA Inception ルートも開ける。健康KPI 組込みはリソース管理として完全支持。

**投票: Go**

### 孫正義
群戦略の本質、各業種で No.1 を作る構造を支持。「Reboot Japan」名称はもっと壮大にすべきと指摘 (ただし出願自体は即実施)。宮野氏のビジョンと狂気は CEO 自身が直接見て判断せよと注文。

**投票: Go (商標名称再検討要望、出願は即実施)**

## 付帯条件 (Go決議の実行条件)

### 即時実施 (今週中)
- 商標「Reboot Japan」シリーズ + 代替候補名 (Japan Reboot / Renaissance Japan / Phoenix Japan / Nihon Saisei 等) の併願出願
- 6月3日 宮野氏 MTG アジェンダを6月1日までに設楽が事前共有

### Partnership Charter 必須条項 (Gates 条件)
1. 収益分配方法 (紹介料/成功報酬/エクイティ持分の三層)
2. 知財帰属 (Claude経営参謀IP / 業種別エージェント / Knowledge Graph は当社100%)
3. Exit条項 (双方向の離脱・解除・紛争解決)
4. 競業避止
5. 役員就任の有無
6. NDA 先行締結

### 速度要件 (Musk 条件)
- Partnership Charter ドラフト 6月10日まで (Schwarzman原案7月15日を前倒し)
- 士業ファネル最初の10件アプローチ 6月末まで着手
- 競合動向月次モニタリング 今月開始

### 名称・KPI 全社適用 (全員支持)
- 「CFO代行 → CFO Copilot」呼称変更を Jobs (ブランディング) と Kotler (マーケティング) 経由で全社適用
- CEO 健康 KPI を月次取締役会で Attia が報告する仕組みを構築。重大悪化シグナル定義 (睡眠/心拍変動/メンタル指標) は Attia が今月中に策定

### 定期報告
- Schwarzman は月次で Roll Up 進捗を取締役会に報告
- 2026-08-27 Month 3 末取締役会で Phase 1 (試行買収) 移行の改めての Stop/Go 決議を取る

## CEOへの最終判断ポイント
1. 宮野友樹氏のビジョン・狂気の有無を6月3日MTGで CEO自身が直接判断
2. Phase 0 の CEO 時間配分 (推奨: 本業 80% / Roll Up 20%) の明示
3. 2027 Series A の資金調達意思を 8月末 Month 3 末取締役会までに決定
4. 商標名称の最終決定 (今週中)
