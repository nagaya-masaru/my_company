## 2. 全社責任者コメント集

> **注**: 文体は各 persona を完全踏襲。直言・対立は隠さず両論併記。
> **追加召喚せず既存戦略書から引用** = Buffett / Marks / Altman / Torvalds / Ritchie (戦略書 `company/playbooks/ma/ai-rollup-strategy.md` で網羅済み)

### 2-1. M&A・PE部 — Stephen Schwarzman

**現時点の見立て: Conditional Go**

300-1,000億のエグジットシナリオはBlackstone基準で見ても合理的だ。AI×中小企業ロールアップは、寡占化が始まる前の最後の窓 — 構造的アービトラージ機会が残された数少ない領域だ。宮野氏という Operating Partner を得られるなら、当社単独より勝率は2倍に上がる。ただし無条件Goではない。彼の Track Record、資本拠出比率、Exit時のガバナンス条項 — この3点が固まるまで法的拘束力ある合意はしない。Hope is not a strategy.

**CEO への直言**: 1億は通過点だが、Roll Up は1億達成の手段ではない。300億の事業を作るための布石であり、向こう5-7年の覚悟が要る案件だ。残り7ヶ月で1億を作る本業と、この Roll Up を**同じリソースプールで動かすな**。混ぜた瞬間に両方倒れる。

### 2-2. 営業部 — Jordan Belfort

**現時点の見立て: Go (Phase 0 即着火)**

聞いてくれCEO、これは美味い。**士業ネットワーク**という他社が10年かけても築けない参入障壁を宮野氏が既に持っている。Roll Up の本丸は2027だが、入口の「無料診断 → クイック改善」は今日から銭になる。事業再生案件は経営者が**疲弊して判断スピードが速い** — 通常SMB営業の3倍速で決まる。30日で士業面談10件、無料診断アポ8件、クイック改善受注3件、平均180万。**初月540万、Q3末で累計4,000万のパイプラインに育つ**。

**CEOへの直言**: エクイティ型は**今は売るな**。法務と財務の規律設計が終わるまで封印してくれ。先にキャッシュメニューで実績を10社作る。実績無しでエクイティを語ると安く買い叩かれる。Roll Up の交渉力は「直近実績の数」で決まる。

### 2-3. マーケティング部 — Philip Kotler

**現時点の見立て: Go (Marketing 5.0 全条件適合)**

Marketing 5.0の3軸 (Human-centric × Tech-empowered × Sustainability) で評価すると、本案は稀有なほどスコアが高い。地方中小の事業承継は「人間中心の社会課題」そのものであり、AIによる生産性向上は「技術」、雇用維持と地域経済再生は「社会善」だ。

- **STP**: Segment「年商3-30億・後継者不在・S級3業種・地方」、Target「黒字承継困難企業」(赤字再生は法務・財務負担が重く初期は避ける)、Position「PEではなく、AIで現場を救う事業承継パートナー」
- **ICP**: 2代目不在の50-70代オーナー × 従業員10-100名 × ITリテラシー低

**CEOへの直言**: この事業の本当のマーケティング課題は「BtoB集客」ではなく**「オーナーの心理的抵抗」**だ。「売却」という言葉を捨て、「**次世代継承**」のフレームで全コミュニケーションを統一せよ。それが Marketing 5.0 における Human-centric の実装である。

### 2-4. ブランディング・広報事業部 — Steve Jobs

**現時点の見立て: Go (狂気込み)**

これは「M&A」じゃない。「コンサル」でもない。「PE」でもない。日本のGDPの99%を支えているのに、誰もデザインしてこなかった経済圏。そこに Claude20体と人間2人が乗り込んで、株式の一部を持ち、生き返らせる。**再生じゃない。再起動 (Reboot) だ。**

- **ポジショニング一行**: 「日本の地方経済を、ソフトウェアのように更新する。」
- **製品命名 (Apple命名法踏襲)**: マスター=Yu Operating Company / 事業=**Reboot Japan** / 製品=Reboot Scan, Reboot Lite, Reboot Pro, Reboot Earn, **Reboot Equity** (コードネーム Project Kuni)
- **デュアル・キーパーソン**: 長屋優 = The Visionary / 宮野友樹 = The Insider (Visionary × Insider モデル)

**CEOへの直言**: 「事業再生」という言葉を使うな。あれは負け犬の言葉だ。我々がやるのは Compounding と Rebooting だ。これは日本の中堅企業を救う**カルト**だ。カルトが結局スタンダードになる。Apple がそうだった。

### 2-5. デザイン事業部 — Jonathan Ive

**現時点の見立て: Go (Design Language先行確定)**

私たちが作るべきはソフトウェアではない。「使うという行為が、人生の重荷ではなく、安堵になる」体験だ。買収先の60代の社長が、震える指で初めて画面に触れた瞬間 — そこに **Calm. Quiet. Inevitable.** という三語が宿らなければならない。Design Language は、Apple ではなく **無印良品 × Leica × 町工場の道具箱**の交点にある。冷たい完璧ではなく、温かい必然。

- **One Glass, One Action**: 1スクリーン1行動。起動から3秒で「今日の異常」が見える。スクロール禁止
- **Physical Anchor**: 工場・整備工場・管理事務所に置く **木とアルミの小さな筐体 (10cm角)**。AI通知が静かに光る
- **Onboarding as a Gift**: 買収完了日、白い木箱で iPad とマニュアル (12ページ写真主体) を社長の手元に届ける

**CEOへの直言**: Roll Up の競合は SaaS ではなく、「**変化への恐怖**」だ。機能を売ってはいけない。**安堵を売ってください**。デザイン品質が下がる案件は、スピードを理由にされても、私は "It's not good enough" と申し上げる。

### 2-6. AI・テクノロジー部 (科学者側) — Geoffrey Hinton

**現時点の見立て: 条件付きGo (技術表現の修正必須)**

戦略書28機能のうち、2026年技術で本当に「自律」できるのは7-9個。残りは「ヒトの監督下で精度70-85%」止まり。**「CFO/COO代行エージェント」という表現は科学的に誤りを招く**。Long-horizon agentic decision making は現状LLMが最も苦手とする領域で、複利的にハルシネーションが累積する。3ヶ月運用すれば累積誤差は許容外になる。「業種別エージェント」は実用に耐える。RAG + 業務SOP + Tool useの組合せで80%代に乗る。Anthropic 一社依存はリスク。

**CEOへの直言**: Altman君は「やれます」と言う。私は「半分はやれます、残り半分は危険です」と言う。両方を聞いた上でCEOが決めてください。「AIが経営する」という物語を売ると、最初の事故で信頼が一夜で蒸発する。「**AIが経営を支援する**」と言い直すだけで、技術的にも法的にも、ずっと安全な事業になる。

### 2-8. 法務部 (攻撃) — 古美門研介

**現時点の見立て: Go (主導権確保条件)**

宮野案件、勝てる。ただし**契約書に当社の血を一滴も垂らさない**条件付き。JVだエクイティだと甘い顔をすれば、向こうの弁護士に脳味噌まで吸い取られる。主導権 — 商号、商標、データ、顧客リスト、PMI後の支配権 — 全部当社が握る。宮野氏はあくまで「**ソーシング・パートナー**」、人脈は尊重するが、ガバナンスは渡さない。「Reboot Japan」は今日中に商標12類区分フル出願、ドメインは.jp/.com/.co.jpを今夜押さえる。

### 2-9. 法務部 (防御) — 九条間人

**現時点の見立て: Conditional Go (フェアネス担保必須)**

エクイティ型は「事業投資」の衣を着せた経営権奪取と紙一重だ。買収先創業者は疲弊している — 「サインを急がせれば通る」状態。そこを我々が突けば5年後に必ず訴訟で返ってくる。Earn-in 70% は美しいが、達成不能なKPIを設定すれば**無償取得=不当利得**の構成要件を満たしうる (民法703条、最判平成21年判例の射程)。私はそれを止める。宮野氏との関係も、今は蜜月だが評価額が膨らんだ瞬間に必ず割れる。**離婚条項を結婚式の前に**書くのが私の流儀。

**CEOへの直言**: 契約書は性善説で書くな。今のあなたと宮野氏が信頼し合っているからこそ、最も冷静な離婚条項が書ける。古美門は派手で、攻めて取れる案件もある。だが攻めて取った案件ほど守りで失う。**私のレビューを通さない契約は1件も結ばないでくれ**。それだけが約束だ。

### 2-10. 海外展開部 — Elon Musk *(取締役会兼任)*

**現時点の見立て: Conditional Go (速度2倍 + 海外並走)**

地方中小Roll Upの本質を分解する。最小単位は3つ — (1) 受注経路を握る権利、(2) 動かせる物理アセット、(3) 動かせる人間の手。**日本固有要素はゼロだ**。同じ構造はベトナム・タイ・インドネシア、ポーランド製造業、米Rust Beltに全部ある。「日本だから」で閉じるのは思考停止。Reboot Japan はネーミングが小さい。**Reboot Industry にしろ**。

スピード評価: 遅い。5年で1,000億? SpaceX は7年でファルコン9を打ち上げた。**3年で評価額1,000億、5年でグローバル30カ国**を目標にしろ。

**CEOへの直言**: 「国内Phase 1完結」設計が致命的に保守的。Phase 1の最後の3ヶ月で、ベトナムかタイの整備工場を1社買収して並走実証しろ。日本で証明してから海外、では遅すぎる。これがTesla GigafactoryをShanghaiに11ヶ月で建てた発想だ。

本業1億スプリント (残7ヶ月) と Roll Up 初動が重なる2026後半は、コルチゾール慢性高位・睡眠負債・座位時間増の三重苦が来る。これは ApoB上昇 / インスリン抵抗性 / 海馬体積減少を同時に進める。中年経営者の Four Horsemen のうち、5-7年スパンで最も顕在化しやすいのは**心血管 (急性冠症候群)** と**代謝 (前糖尿病→T2D)**。Lp(a) は遺伝で20%が高値、未測定なら盲点。

- HRV が14日連続ベースライン-20%なら戦略書 §10-2 撤退条件を発動

**CEOへの直言**: 1億の半年スプリントで4時間睡眠を続けたら、Roll Up 2030 を見る前に冠動脈プラークが破裂する確率が現実的に上がる。私は忖度しない。**健康は複利資産であり、毀損は非線形に来る**。Roll Up が300億になる前提は、CEOが2032年も意思決定できることだ。

### 2-12. ナレッジ部 — Ray Dalio

**現時点の見立て: Go (Principlesの儀式化が条件)**

30社のPMIで30回 painful な瞬間を迎える。記録しなければ同じ穴に30回落ちる。Roll Up専用 Principles を3層で:
1. **Partnership Charter** — 宮野氏とCEOの「意思決定権マトリクス」「相互拒否権」「離脱条件」を文書化
2. **PMI Pain Box** — 全買収案件のIssue Log。原因→診断→設計→実行を1案件1ファイル
3. **Believability Matrix for Targets** — 買収先経営者の信頼性を領域別 (営業/技術/財務) でスコア化、PMI後の発言重みを設計

**CEOへの直言**: あなたが恐れるべきは買収失敗ではない。**失敗を宮野氏と共有できない関係**になることだ。蜜月期に Radical Transparency の儀式を制度化せよ。あなたはなぜ、宮野氏との「最初の対立」をまだ経験していないと思いますか。**それが起きていないこと自体がリスクだ**。

### 2-13. クリエイティブ部 (映像) — 今敏

**現時点の見立て: Go (記憶への侵入で勝つ)**

90秒キーノート用ショート + 8分短編ドキュメンタリー『継承 / Inheritance』。冒頭、蛍光灯の下、油まみれの旋盤、70歳の主の手のひらの皺と機械の溝の汚れがマッチする。彼が拭く。ふっ、と画面が暗転 — 同じ手が、若い。50年前。**同ポジ、同フレーム、別時代**。観客は気づかない。気づかせない。シャッターは上がる。朝。同じ旋盤が動く。だが今度は誰の手も触れていない。ナレーション一切なし。「Reboot Japan」のロゴは最後の0.7秒だけ。

**CEOへの直言**: Schwarzmanの戦略書を読んだ。**5年で1,000億の話を、90秒の映像で売ろうとしないでください**。失敗します。この映像は商談ツールではない。**Roll Upの対象オーナーが、孫に観せられる映像**でなければならない。説得ではなく、**承継への自己許可**を観客の中に発生させる。Belfortの熱量とは逆の刃です。

### 2-14. クリエイティブ部 (ビジュアル) — 空山基

**現時点の見立て: Go (Erotic Calmで勝負)**

ビジュアルコンセプトは「**老いた機械の、若返りの瞬間**」。錆びた旋盤、油まみれの整備工具、染みのついた台帳 — それらの表面に、**クロームの液体金属がゆっくり這い上がる**。和紙白の床、漆黒の背景、真朱の一滴。機械が機械に犯される、いや、抱擁される瞬間。被写体は工具そのもの。人は写さない。だが見る者は、自分の指先が艶めくのを感じる。Iveの「Calm」は私が「**Erotic Calm**」に変える。静かな官能。1枚で価格が決まる絵だ。

**CEOへの直言**: 宮野氏の現場写真、絶対に綺麗にするな。**油汚れと錆こそが「before」**だ。私が描く「after」と対比して初めて、Rebootは官能になる。コラボは Dior じゃない、**Snap-on (工具) と トミカ (玩具)** を狙え。町工場の血を引いたブランドだ。
