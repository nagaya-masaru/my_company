## 主な結論

### 戦略修正 (4つの本質的変更)
1. **商品構造を A/B 二層 → D/A/B/C 四層に再整理**: D層 (ライト版、零細250万社向け漏斗) を追加、C層 (Roll Up) は Phase 0 = 売上計上0で準備のみ。
2. **業種戦略**: S級3業種 (製造業/自動車整備/不動産管理) に集中、士業を入口チャネル化。
3. **言葉の統一**: 「事業再生/コンサル/代行」を全社NGワード化、「次世代継承/Copilot/Assistant」に統一。
4. **取締役会決議の正式組込**: 健康KPIをRoll Up撤退条件に格上げ。CEO時間配分は本業80%/Roll Up20%固定 (7月時点)。

### KPI ツリー再設計
- 北極星: 売上1億 (12/31)
- L1: A/B/D 各層売上 + C層準備指標 (Charter署名/商標出願/士業面談) + 健康指標 + 月次バーン上限
- L2: 営業/デジマ7部門/プロダクト/CS/AI Safety/健康/法務 の各部署ダッシュボード
- レビュー儀式: 日次Slack/週次30分/月次90分/四半期取締役会 (Dalio Principles準拠)

### スケジュール
- **6/30まで**: Week1 (商標確保) → Week2-3 (Partnership + プロダクト基盤) → Week4 (統合リハ + GO/NO-GO判定)
- **7/1〜12/31**: M1 (着火) → M2 (学習&加速) → M3 (単月黒字+Phase 1起動) → M4 (スケール) → M5 (押し込み+試行買収クロージング) → M6 (1億着地+2027 Series A準備)
- 重要イベント: 8/27 Month 3末取締役会 (Phase 1試行買収 Stop/Go決議)、11/26 Phase 2決議、12/26 年次評価

### 撤退条件
15項目に拡張 (Buffett規律 + Attia健康 + Marksサイクル + AI Safety + Marketing漏斗)。CEO HRV/睡眠/血圧の正式組込が特徴。
