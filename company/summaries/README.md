# セッションサマリ

長屋CEOと事業部長エージェントの会話セッションの記録。

## ルール

- **命名**: `yyyymmdd_summary_{topic-slug}.md`
- **保存タイミング**: セッション終了直前、メインClaudeが必ず書く
- **対象**: 実質的なやり取りがあった全セッション (雑談・単発質問はスキップ可)
- **同日複数セッション**: トピックが異なれば別ファイル。同じならファイル末尾に追記

## ファイル構造 (`_template.md` 参照)

1. 議題
2. 主な結論
3. 参加した事業部長エージェント
4. アクションアイテム
5. Decision Log項目

詳細は [CLAUDE.md](../../CLAUDE.md) の「セッションサマリ保存ルール」セクション。
