## 0. プロジェクトコードネームの意味

`ai-roll-up-reboot-japan` ─ この4語に本事業の全戦略思想が圧縮されている。

| 要素 | 意味 | 戦略上の位置づけ |
|------|------|------------------|
| **AI** | AI/DX を実行手段として用いる | 売り物は「AI」ではなく「利益が残る会社への変革」。AIは“見せない武器” |
| **Roll-up** | 改善余地のある中小企業を継続契約で束ね、最終的にエクイティ参画・統合する | 月額伴走 → 成果報酬 → エクイティ型 の階段でロールアップ実現 |
| **Reboot** | 事業再生 / 業務再起動 | 「再生支援」と言わず「利益が残らない構造を立て直す」と翻訳 |
| **Japan** | 地方中小企業 / 日本市場フォーカス | 税理士・士業・地銀・信金ネットワークが武器 |

本計画は **「AI/DXコンサルを起点に、信頼を蓄積し、最終的に企業価値向上+エクイティ参画によってロールアップ型の事業再生プラットフォームを構築する」** ことを目的とする。

## 1. 事業の North Star (本事業限定)

> **資料原文**: 「最初は製造業・建設業・店舗ビジネスの3業種に絞り、1社モデルケースを作る。成果事例ができれば、横展開できる。」

これを基に、本事業の North Star を以下に置く (年商目標ではなく、複利の起点としての成果に置く):

### 1-1. North Star Metric (NSM)
**“黒字伴走社数”** = 月額伴走契約中で、かつ過去90日以内に Before/After で**測定可能な利益改善 (粗利・工数・請求漏れ回収のいずれか)** を実証できた顧問先の数

- フェーズ1終了時 (T+6M): **3社**(モデルケース×3業種)
- フェーズ2終了時 (T+12M): **15社**
- フェーズ3終了時 (T+18M): **40社** + エクイティ参画 **2-3社**

### 1-2. 補助 KPI
| 区分 | 指標 | フェーズ1 | フェーズ2 | フェーズ3 |
|------|------|-----------|-----------|-----------|
| 入口 | 提携士業数 | 5 | 20 | 50 |
| 入口 | 月次紹介数 | 3 | 12 | 30 |
| 受注 | 無料診断 → 月額伴走の転換率 | 30% | 40% | 45% |
| 滞留 | 月額伴走 解約率(月次) | <5% | <3% | <2% |
| 質 | NPS (顧問先) | ≥30 | ≥40 | ≥50 |
| 質 | 月次改善実証率 | 60% | 75% | 85% |

NSMを“契約数”ではなく“**実証付き伴走数**”に置く理由 = ロールアップの素材は「数字で改善できた会社」だけだから。

## 2. 戦略コンセプト(資料の魂を保持)

### 2-1. 売り物は何か
- ❌ AIを売る
- ❌ DXを売る
- ❌ システムを売る
- ✅ **「利益が残る会社」「人手不足でも回る会社」「社長依存をなくした会社」を売る**

### 2-2. 言い換え表 (資料から踏襲)
| NGな言い方 | 刺さる言い方 |
|------------|--------------|
| AI導入しましょう | 社長・役員が抱えている確認作業を毎月30時間削減します |
| DX化しましょう | 紙・Excel・LINEでバラバラな情報を1つにして、誰が見ても回る状態にします |
| システム作ります | まず既存ツールで小さく改善し、費用対効果が見える範囲だけ投資します |
| 効率化します | 人を1人採る前に、今いる人で回る仕組みを作ります |
| 再生支援します | 利益が残らない構造を、原価・人件費・管理工数から立て直します |

### 2-3. 入口は「税理士・士業・金融機関」
資料原文:「税理士・行政書士・司法書士にとって、顧問先の倒産・資金繰り悪化・資料提出遅延は自分たちの収益と業務負担に直結する」

→ 「顧問先をDX」ではなく「**先生の顧問先の月次・資金繰り・業務改善を実行支援し、顧問先の継続可能性を高める**」と訴求する。
→ 士業側にもメリット(月次の早期化、資料回収負担減、追加提案機会)が立つ提案構造を必ず作る。

## 4. 提供メニュー (5層設計 = 資料の収益モデルを踏襲)

| 層 | メニュー | 内容 | 期間 | 価格 | 戦略的狙い |
|----|---------|------|------|------|-----------|
| 入口 | **無料診断** | 業務フロー・資料・数字を確認し、改善余地を洗い出す。Before表 + 30日改善案を提示 | 1-2回 | 無料〜低額(¥0-3万) | モデルケース獲得 / 信頼形成 |
| 体験 | **クイック改善** | 1業務だけ対象に、フォーム化・表整理・テンプレ化 | 2週-1ヶ月 | ¥10万-30万 | 体感価値を出す / 月額への橋渡し |
| 主軸 | **月額伴走** | 複数業務改善 + 社内定着 + KPI管理 + 月次改善会議 | 3-6ヶ月(自動更新) | 月¥20万-80万 | 継続収益化 / ロールアップ素材化 |
| 連動 | **成果報酬** | 削減工数・粗利改善・請求漏れ回収などの成果連動 | 3-12ヶ月 | 改善額の10-30% | 利益改善と報酬を一致 |
| 出口 | **エクイティ型** | 資金余力が乏しいが改善余地が大きい企業に、株式・種類株式等で参画 | 1-3年 | 個別設計 | **企業価値向上と出口収益(ロールアップ実現)** |

### 4-1. メニュー階段の設計原則
1. **下→上に登る摩擦を最小化** ─ 無料診断で出した Before表 をクイック改善でそのまま叩く、月額伴走で月次改善会議の議題にする。同じドキュメントを使い回せる仕組みを作る (後述: ナレッジ部のテンプレライブラリ)
2. **上の層に来る顧客ほどスクリーニングを厳しく** ─ エクイティ型は「**経営者の姿勢=危機感あり / 外部の意見を聞ける**」が満たせない限り受けない (資料の案件選定基準を踏襲)
3. **下の層は薄利・上の層で稼ぐ** ─ 無料診断とクイック改善は意図的に薄利。月額伴走+成果報酬で回収。エクイティ型は10年で複利化

## 6. ユニットエコノミクス(仮置き / Buffett監査前)

### 6-1. 月額伴走 1社あたり

| 項目 | 想定値 |
|------|--------|
| 月額売上 | ¥30万-80万 (平均 ¥50万) |
| 直接原価 (PM工数 / AI APIコスト / SaaS再販コスト) | ¥15-25万 |
| 粗利 | ¥25-30万 / 月 |
| 平均継続月数 (LTV分母) | 18ヶ月(初期前提) |
| LTV (粗利ベース) | ¥450-540万 |
| CAC (紹介経由) | 0-5万円(士業ネット主体) |
| CAC (広告経由) | 30-80万円 (フェーズ2以降の補助チャネル) |
| **LTV/CAC** | 紹介:90倍以上 / 広告:6-15倍 |

→ **紹介経路の経済性が圧倒的**。広告は二の矢として使う。これは戦略的にも「士業ネット主体」が王道であることを支持する。

### 6-2. 成果報酬

- 改善額の 10-30% / 案件
- 平均改善実現額(年):¥500-2000万 (請求漏れ回収・粗利改善等の合算)
- ロイヤリティ売上:¥50-600万 / 顧客 / 年

### 6-3. エクイティ型

- 出資額: ¥500万-3000万 / 案件 (種類株式 / SAFE 相当)
- 出口想定: 5-10年 / IRR 15%以上を最低基準(Marks基準で投資審議)
- リスク: 1案件 = 当社月次バーンの 5-10ヶ月相当 / 個別取締役会決議必須

## 7. 部署別戦略(全部署、本事業へのコミット内容を明文化)

ここから 22 事業部 + 配下 4 本部 (システム開発 / AI開発 / デジタルマーケティング / 取締役会・秘書室) の戦略を一覧する。
本事業は「全社プロジェクト」と位置づけ、各部署が**実行義務を負う**ロールを記す。

### 7-1. 営業部 (Belfort) — sales-belfort

**役割**: 士業ルート開拓・初回面談・クロージング・反論処理 の最終責任

**戦略**:
- 入口は **税理士・行政書士・司法書士**。フェーズ1は5名、フェーズ2で20名、フェーズ3で50名の提携を作る
- ストレートライン:「先生の顧問先で、売上はあるのに利益が残っていない会社はありませんか?」→Yesルート → 顧問先紹介 → 無料診断アポ
- **反論処理マニュアル**(資料の士業悩み表をベースに作成):
  - 「ウチの顧問先はAIなんて必要ない」→「先生、AIではなく **資料回収を1週間早める** 話です」
  - 「顧問先には負担になる」→「**社長の確認作業を月30時間減らす** 話です」
  - 「結果が出るのか?」→「初回30日でBefore/Afterを数字で出します。出なければ撤退します」
- 営業 KPI: 月次アポ20件 → 無料診断10件 → 月額伴走3件 (フェーズ2想定)
- インセンティブ: 月額伴走 初月売上の20% + 12ヶ月継続時に追加ボーナス
- スクリプトは **「AIを売らない / 利益を売る」** を口語化。LINE一行スクリプトを 業種×状況で50本用意

**ヒット商談の標準シナリオ**:
1. 士業ヒアリング(顧問先で困っている社長は?)
2. 紹介依頼 + 1枚提案書送付
3. 三者面談(士業同席のZoom 30分)
4. 無料診断 (Onsite or Zoom 90分 × 1-2回)
5. 30日改善提案 + 月額伴走見積
6. クロージング:「先生もご一緒に月次改善会議に入っていただきます」

### 7-2. 法務部 (古美門研介 / 九条間人) — legal-komikado / legal-kujou

**役割**: 契約構造・成果報酬の法的整理・エクイティ型のリスク管理

**論点と方針**:
| 論点 | 古美門の判断 | 九条の判断 |
|------|-------------|------------|
| 成果報酬契約の合法性 | 改善定義(粗利向上/工数削減/請求漏れ回収)を**事前合意+ベースライン凍結**で明確化。日本では成果連動コンサル契約は問題なし | ただし顧問先の社内データ改ざんで「成果」を作る誘惑が出る。**ベースラインは顧問税理士に署名させる**設計を強制 |
| エクイティ型の証券業法該当性 | 種類株式での個別出資はOK。**ファンド型・複数LP募集型は登録が必要** → フェーズ3まで単独投資に絞る | 倒産近い会社の株を持つと取締役責任が来る。**取締役選任は受けない方針**(オブザーバー権で十分) |
| 個人情報・経営情報 | NDAは**相互/3年/反証責任は持ち出し側**で整える | 士業経由案件は「税理士法の守秘義務」が顧問先データに連動する。**先生側の書面同意を必ず取る** |
| 業務委託契約の偽装請負リスク | 月額伴走は「指揮命令なし・成果物単位」を明文化 | 現場常駐型はNG。**遠隔+月次会議+SaaS提供** の形を死守 |

**契約書テンプレ(フェーズ0 W3 までに作成)**:
1. 無料診断同意書 (NDA含む)
2. クイック改善契約書 (一括 / 期間限定)
3. 月額伴走業務委託契約書 (自動更新 / 解約3ヶ月前通知)

### 7-3. 財務部 (Buffett) — finance-buffett

**役割**: 価格決定 / キャッシュフロー / 投資判断 / ユニットエコノミクスの規律

**Buffett の判断**:
- 「**売上はあるのに利益が残らない会社**」を直す事業 = 我が事業自体も同じ規律を持つこと。本事業の月次粗利率は **50% を死守**(これを下回るメニューは販売停止)
- 月額伴走 ¥20万メニューは**初期割引商品**。フェーズ2以降は最低¥30万に引き上げる
- 成果報酬は「**改善が確定するまでキャッシュインしない**」設計。前受金は取らない。回収サイクル90日以内
- エクイティ型は **本業キャッシュ(=月額伴走+成果報酬)で出資原資を作る**。借入や外部資金は最終手段
- 月次レビュー指標:
  - 売上総利益率
  - 顧客あたり 直接原価
  - SaaS再販コストの ARPU比率(目標 <15%)
  - 1顧客あたり投下工数 (PM工数 vs 自動化工数の比率)

**規律**:
- 「月額¥80万のメニューを売っているのに、AI APIコストで月¥30万使っている」状態は**即停止**
- 1事業の月次バーンが月次粗利を超えたら即時CEOアラート

### 8-2. 撤退基準

以下のいずれかに該当した場合、取締役会で撤退/ピボット決議:
- フェーズ1終了時 (T+6M) に**黒字伴走社数 < 2社**
- 月額伴走の**解約率 > 10%/月 が3ヶ月連続**
- **粗利率 < 30% が3ヶ月連続**
- 顧問先データ漏洩等の**重大インシデント発生**
- CEO の健康問題で6週間以上の活動停止
